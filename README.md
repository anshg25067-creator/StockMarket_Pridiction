# StockMarket_Pridiction
Introduction:-

This is my second official web development project, focused on the stock market, specifically for intraday trading.
The main idea behind this project is to create a simple web application that allows users to search stocks, view their recent data, and get basic insights that can help in short-term trading decisions.
This project is more about learning how real-world financial data works and how it can be used in a web application.

The purpose of this project:-
To understand how stock market APIs work
To build a practical application using real-time data
To explore how basic prediction or trend analysis can be implemented
To create a tool that can assist intraday traders with quick insights

API used:-

I have used Marketstack API to fetch stock market data.

Features:-
* Search stocks using ticker symbols (e.g: AAPL, TSLA)
* Display current stock price and key details
* Sort stocks based on price or performance
* Filter stocks
* Add basic prediction logic for short-term trends

Tech Stack:-
HTML
CSS
JavaScript

I planed to use API to get data of open and close price of particular stock and, use few math techniques to calculate whether stock will go up or down.
